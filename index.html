Ce plugin gere votre programme tv

Le plugin progtv est un parser XML de fichier EPG genere par le site www.kazer.org.
Pour utiliser ce plugin il vous faut une clef unique fourni par kazer lors de votre inscription sur le site. Ne vous inquiétez par
l'inscription se fait en 20 secondes et est gratuite. elle est nécéssaire pour determiner vos favoris tout simplement. Avec ce plugin il vous suffit de demander a SARAH le programme télé de pour une chaine, pour un jour et/ou un soit de votre choix.
Consulter la liste des commandes pour plus de détails.

    Enregistrer vous sur le site www.kazer.org pour obtenir votre clef kazer.
    Rendez vous dans l'onglet CRON de SARAH, afficher le panneau de configuration du plugin et rentrez votre kazeruserkey (sans espace avant ni apres).

  • Fonction programme une seul chaine:
    Cette fonction permet de vocaliser le programme tele pour une chaine et un jour de votre choix.
    - Sarah, programme tele M6 lundi soir.
    - Sarah, programme tele M6 maintenant

  • Fonction liste programme toutes chaines:
    Cette focntion permet d'obtenir et de vocaliser la liste des programmes télé pour l'ensemble des chaines presentes dans votre liste kazer.
    - Sarah, liste des programmes lundi soir.

  • Fonction liste programme pour chaines favorites:
    Cette focntion permet d'obtenir et de vocaliser la liste des programmes télé uniquement pour les favoris que vous aurez renseigné dans le fichier progtv.json
    - Sarah, liste des programmes favoris lundi soir.

  • Fonction recherche categorie:
    Cette fonction permet de recherche les chaines tele et de vocaliser le nom du programme pour une categorie souhaitée.
    - Sarah, recherche categorie film dimanche soir.

  • Integration plugin AskMe et XBMC:
    Lors de l'activation de la focntion "programme tele", le plugin askme est activé pour vous permettre d'executer deux actions:.
    . fonction "afficher": en lien avec le plugin xbmc, cette fonction permet de switcher sur la chaine que vous avez consulté.
    . fonction "description": vocalise la description du programme.

    dites simpelment "afficher" ou "description" apre le messsage vocal "que veux tu faire?"
