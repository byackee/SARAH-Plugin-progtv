Ce plugin gère votre programme tv

Le plugin progtv est un parser XML de fichier EPG généré par le site www.kazer.org. Pour utiliser ce plugin il vous faut une clé unique fournie par kazer lors de votre inscription sur le site. Ne vous inquiétez pas l'inscription se fait en 20 secondes et est gratuite. Elle est nécessaire pour déterminer vos chaines préférées.
Avec ce plugin il vous suffit de demander à SARAH le programme télé pour une chaine, pour un jour et/ou un soir de votre choix.

Consulter la liste des commandes pour plus de détails.

Enregistrez vous sur le site www.kazer.org pour obtenir votre clé kazer.
Une fois enregistré sur le site, allez dans "Mes Chaines" et la clé est le userhash (au début de la page).
Rendez vous dans l'onglet CRON de SARAH, puis affichez le panneau de configuration du plugin et rentrez votre kazeruserkey (sans espace avant ni après).

  • Avoir le programme pour une seul chaine :
    Cette fonction permet de vocaliser le programme télé pour une chaine et un jour de votre choix.
    • Sarah, programme télé M6 lundi soir.
    • Sarah, programme télé M6 maintenant
  • Avoir la liste de tous les programmes pour toutes les chaines :
    Cette fonction permet d'obtenir et de vocaliser la liste des programmes télé pour l'ensemble des chaines présentes dans votre liste kazer (dans "Mes Chaines"). A noter que si vous n'avez pas ajouté des chaines, il en prendra par défaut.
    • Sarah, liste des programmes lundi soir.
  • Avoir la liste des programme pour ses chaines favorites :
    Cette fonction permet d'obtenir et de vocaliser la liste des programmes télé uniquement pour les favoris que vous aurez renseignés dans le fichier progtv.json
    • Sarah, liste des programmes favoris lundi soir.
  • Rechercher un programme dans une catégorie :
    Cette fonction permet de rechercher les chaines télé et de vocaliser le nom du programme pour une catégorie souhaitée.
    • Sarah, recherche catégorie film dimanche soir.

  • Intégration du plugin AskMe :
    Lors de l'activation de la fonction "programme télé", le plugin askme est activé pour vous permettre d'exécuter deux actions lorsque SARAH vous pose la question "que veux-tu faire ?" :
    • dites "description" : vocalise la description du programme.
    • dites "afficher" : si vous utilisez le plugin XBMC, alors cette fonction permet de zapper sur la chaine que vous avez consultée.
